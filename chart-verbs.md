# 📊 图表类动词表达整理（IELTS Task 1）

在 IELTS 小作文 Task 1 图表题中，我们常常需要描述图表“展示了什么”，此类动词是开头句的必备。

本页整理常见图表描述动词，助你写出准确、书面、得体的第一句。

---

## 📝 常见图表类动词列表

| 动词 | 中文释义 | 示例句型 |
|------|----------|----------|
| show | 展示，显示 | The chart shows the changes in birth rates. |
| illustrate | 说明，阐明 | The bar graph illustrates the data clearly. |
| present | 呈现，展示 | The table presents information about global sales. |
| depict | 描绘，刻画 | The pie chart depicts the distribution of energy sources. |
| describe | 描述 | The diagram describes how electricity is produced. |
| compare | 对比 | The chart compares the income levels in four countries. |
| represent | 代表，反映 | The graph represents the population growth trends. |
| provide | 提供 | The chart provides data on unemployment rates. |
| outline | 概述 | The diagram outlines the stages of the process. |
| summarize | 总结，概括 | The table summarizes the number of visitors by region. |

---

## ✨ 实用句型模板

- **The chart shows…**  
  用于开头句最基础的描述。

- **The graph illustrates changes in…**  
  用于数据趋势型。

- **The table presents data about…**  
  适用于表格类图表。

- **The diagram outlines the process of…**  
  专用于工序/流程图。

---

## 🎯 使用建议

- 开头句建议使用动词 + 图表对象 + 时间范围结构，例如：  
  *The bar chart illustrates the sales figures of five companies from 2010 to 2020.*

- 避免重复使用 “show” 一词，多样替换可加分。

---

## 📚 同义词替换建议

| 常用词 | 替代表达 |
|--------|-----------|
| show | illustrate / present / depict |
| describe | outline / summarize |
| compare | contrast / evaluate |
